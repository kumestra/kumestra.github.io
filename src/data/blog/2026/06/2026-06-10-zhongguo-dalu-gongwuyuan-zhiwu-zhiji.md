中国大陆公务员系统里的“职务”和“职级”，最容易让人困惑的地方，是它们听起来都像“级别”。但如果只记一句话，可以这样理解：

**职务看岗位和权责，职级看等级和待遇通道。**

更细一点说，公务员制度里至少有三层概念需要分开：

1. **领导职务层次**：从国家级正职到乡科级副职，是一套从高到低的层级框架。
2. **具体职务**：某个人实际担任的岗位，比如科技部部长、江苏省省长、某县县长、某处处长。
3. **职级**：与领导职务并行的等级序列，主要用于拓展晋升空间，并和工资、住房、医疗等待遇相关。

把这三层拆开之后，很多看似绕口的问题就清楚了。

## 先看职务：层次不是岗位本身

公务员的领导职务层次可以从高到低理解为十个层次：

| 领导职务层次 | 常见说法 |
| --- | --- |
| 国家级正职 | 正国级 |
| 国家级副职 | 副国级 |
| 省部级正职 | 正省级、正部级 |
| 省部级副职 | 副省级、副部级 |
| 厅局级正职 | 正厅级、正局级 |
| 厅局级副职 | 副厅级、副局级 |
| 县处级正职 | 正县级、正处级 |
| 县处级副职 | 副县级、副处级 |
| 乡科级正职 | 正科级 |
| 乡科级副职 | 副科级 |

这些名称不是一个个具体职位，而是**职务层次**。每个层次下面可以有很多具体岗位。

比如，“省部级正职”这个层次下面，可以包括国务院组成部门的部长、省级人民政府省长、省委书记、直辖市市长等。科技部部长和江苏省省长是两个完全不同的具体职务：一个在中央部委，一个在地方政府；一个负责科技行政事务，一个主持省政府工作。但在领导职务层次上，它们通常都属于省部级正职。

所以更准确的表达是：

> 科技部部长和江苏省省长是不同职务，但属于同一领导职务层次。

类似地，“县处级正职”不是一个具体岗位，而是一层。县长、区长、省厅某处处长、地级市某局局长，可能都落在这个层次或相近管理框架中，具体还要看机关规格和岗位设置。

## 再看职级：不是领导岗位，而是并行通道

2019 年施行的《公务员职务与职级并行规定》把“职级”定义为与领导职务并行的晋升通道。它体现政治素质、业务能力、资历贡献，并作为确定工资、住房、医疗等待遇的重要依据。关键点是：**职级本身不具有领导职责**。

这就是为什么一个人可以“没有当处长”，但有比较高的职级；也可以“担任某个领导职务”，同时兼任某个职级。

综合管理类公务员的职级序列，从高到低大致是：

| 层级组 | 综合管理类职级 |
| --- | --- |
| 巡视员 | 一级巡视员、二级巡视员 |
| 调研员 | 一级调研员、二级调研员、三级调研员、四级调研员 |
| 主任科员 | 一级主任科员、二级主任科员、三级主任科员、四级主任科员 |
| 科员 | 一级科员、二级科员 |

这里的“一级调研员”“二级主任科员”容易被误读成某个岗位。其实它们主要是职级名称，不等同于“调研岗位负责人”或“科室主任”。例如，一级调研员可能享受较高的待遇层次，但不自动意味着他是处长，也不自动改变原来的工作职位和领导指挥关系。

## 职务层次和职级有对应关系，但不能简单画等号

在综合管理类公务员中，规定列出了厅局级以下领导职务对应的最低职级：

| 领导职务层次 | 对应的综合管理类最低职级 |
| --- | --- |
| 厅局级正职 | 一级巡视员 |
| 厅局级副职 | 二级巡视员 |
| 县处级正职 | 二级调研员 |
| 县处级副职 | 四级调研员 |
| 乡科级正职 | 二级主任科员 |
| 乡科级副职 | 四级主任科员 |

这个表很有用，但也最容易被过度理解。

“对应”不等于“相同”。一个二级调研员不等于县长或处长；一个四级调研员不等于副县长或副处长。职级可以影响待遇，但不自动带来领导岗位、政治待遇或工作待遇。反过来，担任领导职务的人，也可能按照规定兼任职级，并按照就高原则享受有关待遇。

一句话说：

> 职务和职级有待遇上的连接，但权责上不能混为一谈。

## 再加一个“级别”：工资级别不是同一个词

公务员制度里还有“级别”这个概念。比如某个职级会对应若干公务员级别区间：一级巡视员对应十三级至八级，二级巡视员对应十五级至十级，一级科员对应二十六级至十八级。

这又是另一层。日常理解时可以先这样区分：

| 概念 | 回答的问题 |
| --- | --- |
| 具体职务 | 你担任什么岗位？ |
| 领导职务层次 | 这个岗位处在哪一层？ |
| 职级 | 你在并行晋升通道中的等级是什么？ |
| 级别 | 工资和相关管理中对应的更细等级区间是什么？ |

很多讨论把“级别”“职级”“行政级别”混着说，是因为它们在待遇和组织管理上确实有关联。但在制度文本里，它们不是同一个东西。

## 用几个例子把概念钉住

**例子一：科技部部长和江苏省省长**

这是两个不同的具体职务。前者是中央国家机关中国务院组成部门的正职领导，后者是省级人民政府正职领导。但它们通常都属于省部级正职这一领导职务层次。

**例子二：某省科技厅厅长**

“省科技厅厅长”是具体职务。它通常属于厅局级正职。按照职务层次看，它低于省部级正职；按照具体工作看，它负责的是省级科技行政部门。

**例子三：一级调研员**

“一级调研员”是职级，不是具体领导岗位。一个人是一级调研员，说明他在综合管理类职级序列里处于较高位置，但不能直接推出他担任某个处长、局长或县长职务。

**例子四：科长和一级主任科员**

“科长”通常是具体职务，表示这个人负责某个科室，具有相应管理职责。“一级主任科员”是职级，强调等级和待遇通道。一个人可能是科长，也可能兼任一级主任科员；也可能不是科长，但职级是一级主任科员。

## 为什么要有职务与职级并行

传统上，很多人把晋升理解成“必须当领导”。但领导岗位数量有限，如果所有激励都绑在领导职务上，就容易形成单一的晋升挤压。

职务与职级并行的制度逻辑，是让公务员除了竞争领导岗位之外，也可以通过职级获得职业发展和相应待遇。这对于专业化、长期在一线或业务岗位工作的公务员尤其重要。

也就是说，职级并不是“虚职”，也不是“头衔装饰”。它是一条正式的制度化通道。但它和领导职务的区别也必须清楚：职级可以带来待遇上的变化，却不当然改变岗位权力。

## 最简记忆法

如果要快速判断一个称呼属于哪一类，可以问三个问题：

| 看到的名称 | 判断方法 |
| --- | --- |
| 科技部部长、江苏省省长、县长、处长 | 这是具体职务 |
| 省部级正职、厅局级副职、县处级正职 | 这是领导职务层次 |
| 一级巡视员、二级调研员、一级主任科员 | 这是职级 |
| 十三级、十八级、二十六级 | 这是公务员级别 |

最终可以压缩成一句话：

> 中国大陆公务员的职务体系，先有从高到低的领导职务层次；每个层次下面有许多具体岗位；同时还有一套与领导职务并行的职级序列，用来承载晋升和待遇。

理解了这一点，再看“正部级”“正厅级”“一级调研员”“主任科员”这些说法，就不会把岗位、层次和待遇通道揉成一团了。
